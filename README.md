# Decoded Solutions Website

Professional IT consulting and software development for small businesses and underserved communities.

## Structure
- `index.html` - Homepage
- `about.html` - About Us
- `services.html` - Services
- `portfolio.html` - Portfolio
- `contact.html` - Contact
- `css/style.css` - Styles
- `js/main.js` - Interactivity
- `assets/` - Images and logo

## Deployment
1. Push to GitHub repo: `decodedsolutionsite`
2. Go to Settings > Pages
3. Select `main` branch and `/` (root)
4. Save - Site live at `https://ngbontsi.github.io/decodedsolutionsite/`

## Tech Stack
- HTML5, CSS3, Vanilla JavaScript
- Tailwind-inspired custom CSS
- GitHub Pages hosting (Free)
- Mobile-first responsive design
